# Suchir Bobbili — Portfolio

A single-page, terminal/hacker-themed portfolio built with plain HTML, CSS, and JavaScript (no build step, no dependencies).

## Sections
- **Hero** — animated terminal boot sequence intro
- **About** — background & quick stats
- **Skills** — security & networking, languages, tools
- **Projects** — 6 Python security tools (SOC log analyzer, phishing detector, malware sandbox, reverse shell lab, brute-force simulator, port scanner) shown as an `nmap`-style scan table
- **Experience** — student council leadership & ethical hacking bootcamp
- **Certifications & Education**
- **Contact** — email, phone, GitHub

## Running locally
Just open `index.html` in a browser — no build tools required.

## Deploying with GitHub Pages
1. Push this repo to GitHub.
2. Go to **Settings → Pages**.
3. Set source to the `main` branch, root folder.
4. Your site will be live at `https://<username>.github.io/<repo-name>/`.

## Files
- `index.html` — page structure & content
- `style.css` — all styling (CSS variables at the top for easy theming)
- `script.js` — terminal boot animation, mobile nav, scroll-spy nav highlighting
